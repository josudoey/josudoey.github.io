# NodeJS

## NPM

- [js-jieba](https://www.npmjs.com/package/js-jieba): 結巴分詞 (Jieba) 的 JavaScript 移植版，支援 Node.js 與瀏覽器環境的繁簡中文斷詞處理
- [zbar-qr](https://www.npmjs.com/package/zbar-qr): 基於 WebAssembly 與 ZBar C/C++ 函式庫編譯的條碼與 QR Code 掃描解析工具，高效且無其他相依性
- [promise-singleflight](https://www.npmjs.com/package/promise-singleflight): 實作 Go 語言的 singleflight 模式，防止多個相同 Key 的非同步 (Promise) 任務重複且同時發起執行
- [promise-solo](https://www.npmjs.com/package/promise-solo): 提供序列化與排他性執行的 Promise 包裝器，確保特定非同步任務在同一時間僅有一個在執行
- [composable-hooks](https://www.npmjs.com/package/composable-hooks): 用於組合和管理 Hooks/Middleware 的輕量級 JavaScript 工具庫，便於構建可複用與可擴充的非同步生命週期邏輯

## Example

- [cropper](https://github.com/josudoey/webtool-cropper/): 網頁端圖片裁剪 (Image Cropper) 功能的實作範例與應用
- [ecc](https://github.com/josudoey/ecc): 橢圓曲線密碼學 (Elliptic Curve Cryptography) 的演算法實作與概念驗證範例
- [rsa](https://github.com/josudoey/rsa): RSA 非對稱加密與解密演算法的實作範例及加解密遊樂場

## Web Tool

- [PagePeer](https://josudoey.github.io/PagePeer/): 免安裝瀏覽器端 P2P 檔案與訊息傳輸工具，使用 WebRTC 技術，掃描 QR Code 即可在不同裝置間安全、即時地傳輸文字與檔案，無需伺服器中轉
- [WebRTC](https://josudoey.github.io/web-rtc-sample): WebRTC 點對點 (P2P) 連線與多媒體串流傳輸的基礎實作範例
- [Qrcode](https://josudoey.github.io/webtool-qrcode/): 線上 QR Code 條碼產生器，支援自訂尺寸與內容
- [Qrcode Scan](https://josudoey.github.io/webtool-zbar-qr/): 線上 QR Code 條碼掃描與解析工具 (基於 zbar-qr)
- [json2table](https://josudoey.github.io/webtool-json2table/): 將 JSON 格式資料快速轉換成可視化、可排序與搜尋的互動式表格工具
- [json-schema-generator](https://josudoey.github.io/webtool-json-schema-generator/): 線上根據輸入的 JSON 樣本自動產生對應 JSON Schema 結構定義的工具

## Webgame

- [瞬間記憶大挑戰](https://josudoey.github.io/webgame-memory/): 考驗玩家瞬間記憶力與反應能力的網頁益智小遊戲
- [翻翻樂](https://josudoey.github.io/webgame-matching/): 經典的卡牌配對記憶網頁遊戲 (Matching Game)
